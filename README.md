# Анализ факторов популярности фильмов IMDb и прогнозирование рейтингов

Учебный проект по машинному обучению: анализ и моделирование рейтингов фильмов на IMDb на основе открытого датасета IMDb Top 1000. 
Цель проекта — определить, какие характеристики фильмов влияют на их рейтинг, и построить модель, которая прогнозирует оценку фильма.

## Загрузка данных
- Датасет: imdb_top_1000.csv
- Признаки: Title, Genre, Runtime, Votes, IMDB_Rating
- Ссылка на датасет: [https://www.kaggle.com/datasets/harshitshankhdhar/imdb-dataset-of-top-1000-movies-and-tv-shows]

Архитектура проекта:

Загрузка данных (imdb_top_1000.csv) →
  
→ Очистка и предобработка (удаление пропусков, обработка жанров, преобразование runtime, формирование Votes)

→ Разведочный анализ (распределение рейтингов, топ жанров, зависимость Runtime)

→ Подготовка признаков (OneHotEncoding Genre, числовые признаки)

→ Pipeline моделей (Linear Regression, Ridge Regression, RandomForest Regressor)

→ Оценка качества (MAE, RMSE, R²)

→ Интерпретация результатов (Feature Importances) → Итоговые выводы (лучшая модель, важнейшие факторы)

## Результаты моделей

| Модель                 | MAE   | RMSE  | R²    |
|------------------------|-------|-------|-------|
| Linear Regression      | 0.182 | 0.226 | 0.405 |
| Ridge Regression       | 0.186 | 0.229 | 0.388 |
| RandomForest Regressor | 0.068 | 0.086 | 0.915 |

Лучшей моделью оказался **RandomForest Regressor**, объясняющий ~91,5% вариации рейтингов IMDb.

## Анализ важности признаков (RandomForest)
- Votes — главный фактор
- Genre — умеренное влияние
- Runtime — незначительное влияние
- Визуализация важности признаков строится графиком с процентным вкладом каждого признака в модель

## Выводы
- Лучшая модель — RandomForest
- Количество голосов (Votes) является ключевым фактором рейтинга
- Жанр и длительность фильма также влияют на итоговую оценку
- Линейные модели показали меньшее качество предсказаний, RandomForest лучше работает с категориальными и числовыми признаками одновременно
